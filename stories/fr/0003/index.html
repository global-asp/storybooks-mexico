Les vêtements d’école Ropa de escuela School clothes

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini, Ingrid Schechter

Alexandra Danahy

Monique Bournot-Trites

francés

Nivel 1

Cette robe est longue.

Este vestido es largo.

This dress is long.

Ce pull est grand.

Este suéter es grande.

This jersey is big.

Ce sac est grand.

Este bolso es grande.

This bag is big.

Cette ceinture est…

Este cinturón es…

This belt is…

Ce chapeau est petit.

Este sombrero es pequeño.

This hat is small.

Les chaussettes sont courtes.

Los calcetines son cortos.

The socks are short.

Mais ces souliers sont neufs.

Pero estos zapatos son nuevos.

But these shoes are new.

… ils font bien.

… Me quedan bien.

… they fit well.

Lectura en voz alta: Monique Bournot-Trites
